## usage (local debug)
* 【install npm, yarn】
1. npm install --global expo-cli
2. npm install --global yarn
3. git clone https://github.com/sohrin/liiist.git
4. cd liiist
5. yarn
* yarn run web
* yarn run android
* yarn run ios
